# <a name="labor-cost-tracking-on-projects"></a><span data-ttu-id="97480-103">Registrazione del costo di manodopera nei progetti</span><span class="sxs-lookup"><span data-stu-id="97480-103">Labor cost tracking on projects</span></span>

<span data-ttu-id="97480-105">Dynamics 365 Project Operations registra le stime e le spese di manodopera con la granularità più bassa richiesta nel piano di progetto.</span><span class="sxs-lookup"><span data-stu-id="97480-105">Dynamics 365 Project Operations tracks labor estimates and spend at the lowest required granularity on the project plan.</span></span> <span data-ttu-id="97480-106">La stima finanziaria del costo di manodopera si basa sul lavoro richiesto pianificato e sulle risorse generiche o denominate assegnate a ciascuna attività del nodo foglia nel piano di progetto.</span><span class="sxs-lookup"><span data-stu-id="97480-106">The financial estimate of labor cost is based on the planned effort and the generic or named resources that are assigned to each leaf-node task in the project plan.</span></span> <span data-ttu-id="97480-107">Quando il progetto ha inizio e le persone cominciano a segnalare il tempo di varie attività del progetto, viene riepilogata la spesa effettiva di manodopera che avvia un calcolo delle proiezioni.</span><span class="sxs-lookup"><span data-stu-id="97480-107">When the project begins and people start to report time on various project tasks, the actual spend on labor is summarized which starts a calculation of the projections.</span></span>

## <a name="labor-cost-tracking-view"></a><span data-ttu-id="97480-108">Visualizzazione Registrazione costo di manodopera</span><span class="sxs-lookup"><span data-stu-id="97480-108">Labor Cost Tracking view</span></span>

<span data-ttu-id="97480-109">Nella pagina **Progetti**, nella scheda **Registrazione**, puoi selezionare **Registrazione** > **Costo** per aprire la visualizzazione **Registrazione costi** e vedere l'avanzamento della spesa di manodopera in ogni attività nel piano di progetto.</span><span class="sxs-lookup"><span data-stu-id="97480-109">On the **Projects** page, on the **Tracking** tab, you can select **Tracking** > **Cost** to open the **Cost Tracking** view and see the progress of labor spend on each task in the project plan.</span></span> <span data-ttu-id="97480-110">Questa visualizzazione confronta anche il costo di manodopera effettivo speso per un'attività al costo di manodopera pianificato dell'attività.</span><span class="sxs-lookup"><span data-stu-id="97480-110">This view also compares the actual labor cost spent on a task to the task's planned labor cost.</span></span> <span data-ttu-id="97480-111">Project Operations utilizza le seguenti formule per calcolare le metriche del costo di manodopera:</span><span class="sxs-lookup"><span data-stu-id="97480-111">Project Operations uses the following formulas to calculate labor cost metrics:</span></span>

- <span data-ttu-id="97480-112">**Costo pianificato**: costo stimato di tutte le assegnazioni di risorse in ciascuna attività del nodo foglia</span><span class="sxs-lookup"><span data-stu-id="97480-112">**Planned cost**: Estimated cost of all resource assignments on each leaf node task</span></span>
- <span data-ttu-id="97480-113">**Costo effettivo**: somma di tutti i valori effettivi del costo per il tempo registrato nell'attività</span><span class="sxs-lookup"><span data-stu-id="97480-113">**Actual Cost**: Sum of all cost actuals for time recorded on the task</span></span>
- <span data-ttu-id="97480-114">**Percentuale consumo costi**: costo effettivo ÷ stima costo al completamento</span><span class="sxs-lookup"><span data-stu-id="97480-114">**Cost consumption percentage**: Actual cost ÷ Cost estimate at complete</span></span>
- <span data-ttu-id="97480-115">**Costo rimanente**: stima costo al completamento - costo effettivo</span><span class="sxs-lookup"><span data-stu-id="97480-115">**Remaining Cost**: Cost estimate at complete  – Actual cost</span></span>
- <span data-ttu-id="97480-116">**Costo al completamento**: costo rimanente + costo effettivo</span><span class="sxs-lookup"><span data-stu-id="97480-116">**Cost at Complete**: Remaining cost + Actual cost</span></span>
- <span data-ttu-id="97480-117">**Scostamento costo**: costo pianificato - costo stima al completamento</span><span class="sxs-lookup"><span data-stu-id="97480-117">**Cost variance**: Planned cost – Cost estimate at complete</span></span>

<span data-ttu-id="97480-118">Ogni attività mostra una proiezione dello scostamento costo nell'attività.</span><span class="sxs-lookup"><span data-stu-id="97480-118">Each task shows a projection of the cost variance on the task.</span></span> <span data-ttu-id="97480-119">Se la stima del costo al completamento è superiore al costo pianificato, si prevede che l'attività supererà il budget.</span><span class="sxs-lookup"><span data-stu-id="97480-119">If the cost estimate at complete is more than the planned cost, the task is projected to go over budget.</span></span> <span data-ttu-id="97480-120">Se la stima del costo al completamento è inferiore al costo pianificato, si prevede che l'attività non supererà il budget.</span><span class="sxs-lookup"><span data-stu-id="97480-120">If the cost estimate at complete is less than the planned cost, the task is projected to finish under budget.</span></span>

>
> - <span data-ttu-id="97480-125">La **scheda Stime** calcola il costo stimato utilizzando la stessa valuta del tasso di costo nel listino prezzi.</span><span class="sxs-lookup"><span data-stu-id="97480-125">The **Estimates tab** calculates the estimated cost by using the same currency of the cost rate in the price list.</span></span> <span data-ttu-id="97480-126">Quindi, il costo stimato nella valuta del listino prezzi viene convertito nel costo stimato nella valuta del costo del progetto.</span><span class="sxs-lookup"><span data-stu-id="97480-126">Then, the estimated cost in the price list currency is converted to the estimated cost in the project's cost currency.</span></span> <span data-ttu-id="97480-127">Il costo stimato nella valuta del progetto viene mostrato arrotondato a 2 cifre decimali.</span><span class="sxs-lookup"><span data-stu-id="97480-127">The estimated cost in the project currency is shown rounded to 2 decimal places.</span></span> <span data-ttu-id="97480-128">In nessun momento durante questa conversione viene applicata la precisione della valuta.</span><span class="sxs-lookup"><span data-stu-id="97480-128">At no point during this conversion is currency precision applied.</span></span>

   ></ol>

## <a name="reprojecting-costs-on-leaf-node-tasks"></a><span data-ttu-id="97480-133">Riproiezione dei costi nelle attività del nodo foglia</span><span class="sxs-lookup"><span data-stu-id="97480-133">Reprojecting costs on leaf node tasks</span></span>
